# Expense Tracker PWA

Простое и удобное PWA приложение для учёта расходов и доходов.

## Особенности
- **PWA**: Можно установить на телефон или рабочий стол.
- **Offline**: Работает без интернета благодаря Service Worker.
- **LocalStorage**: Данные сохраняются локально в браузере.
- **Дизайн**: Современный и адаптивный интерфейс.

## Установка
1. Склонируйте репозиторий.
2. Откройте `index.html` в браузере или используйте любой локальный сервер.
3. В браузере появится иконка "Установить" (в строке адреса на десктопе или в меню на мобильном).

## Разработка
Для запуска тестов:
```bash
npm install
npx playwright test
```
